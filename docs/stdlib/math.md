# Math Module

Mathematical functions and constants.

## Functions

| Function | Type Signature | Description |
| :--- | :--- | :--- |
| `abs` | `(x: Integer) -> Integer` | Returns the absolute value of an integer. |
| `sqrt` | `(x: Float) -> Float` | Returns the square root of a float. |
| `pow` | `(base: Float, exp: Float) -> Float` | Returns `base` raised to the power of `exp`. |
| `sin` | `(x: Float) -> Float` | Sine of x (radians). |
| `cos` | `(x: Float) -> Float` | Cosine of x (radians). |
| `tan` | `(x: Float) -> Float` | Tangent of x (radians). |
| `floor` | `(x: Float) -> Float` | Returns the largest integer less than or equal to x. |
| `ceil` | `(x: Float) -> Float` | Returns the smallest integer greater than or equal to x. |
| `round` | `(x: Float) -> Float` | Rounds a float to the nearest integer. |
| `log` | `(x: Float) -> Float` | Natural logarithm (base e). |
| `log10` | `(x: Float) -> Float` | Logarithm base 10. |
| `exp` | `(x: Float) -> Float` | Exponential function (e^x). |
| `min` | `<T>(a: T, b: T) -> T` | Returns the smaller of two values. |
| `max` | `<T>(a: T, b: T) -> T` | Returns the larger of two values. |
| `to_int` | `(x: Float) -> Integer` | Converts a float to an integer (truncates). |
| `to_float` | `(x: Integer) -> Float` | Converts an integer to a float. |
